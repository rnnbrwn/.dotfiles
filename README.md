- `~/.zshrc` is symbolically linked to `~/.dotfiles/zshrc`. To setup a "symlink" in the future, check the Symbolic Links note below.

### Symbolic Links

To create a symbolic link: `ln -s ~/.dotfiles/zshrc ~/.zshrc`
